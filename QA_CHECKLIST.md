## 1. Backend API 테스트 (18개 항목)

### 1.1 인증 (Authentication)

#### 회원가입 (Signup)
- [ ] **1.1.1** 유효한 이메일, 사용자명, 비밀번호로 회원가입 성공
  - **엔드포인트:** `POST /auth/signup`
  - **테스트 데이터:**
    ```json
    {
      "email": "qa-test@example.com",
      "username": "qauser",
      "password": "TestPass123!"
    }
    ```
  - **예상 결과:** 201 Created, 사용자 ID 반환

- [ ] **1.1.2** 중복 이메일로 회원가입 실패
  - **테스트:** 1.1.1의 이메일로 다시 회원가입 시도
  - **예상 결과:** 400 Bad Request, "Email already registered"

- [ ] **1.1.3** 잘못된 이메일 형식으로 회원가입 실패
  - **테스트 데이터:** `email: "invalid-email"`
  - **예상 결과:** 422 Unprocessable Entity

- [ ] **1.1.4** 짧은 비밀번호로 회원가입 실패 (8자 미만)
  - **테스트 데이터:** `password: "short"`
  - **예상 결과:** 422 Unprocessable Entity

#### 로그인 (Login)
- [ ] **1.2.1** 올바른 이메일과 비밀번호로 로그인 성공
  - **엔드포인트:** `POST /auth/login`
  - **테스트 데이터:** 1.1.1에서 생성한 계정
  - **예상 결과:** 200 OK, `access_token` 및 `user` 정보 반환

- [ ] **1.2.2** 존재하지 않는 이메일로 로그인 실패
  - **테스트 데이터:** `email: "nonexistent@example.com"`
  - **예상 결과:** 401 Unauthorized

- [ ] **1.2.3** 잘못된 비밀번호로 로그인 실패
  - **테스트 데이터:** 올바른 이메일 + 잘못된 비밀번호
  - **예상 결과:** 401 Unauthorized

#### JWT 토큰
- [ ] **1.3.1** 유효한 JWT 토큰으로 인증된 엔드포인트 접근 성공

  - **예상 결과:** 200 OK

- [ ] **1.3.2** JWT 토큰 없이 인증된 엔드포인트 접근 실패
  - **테스트:** `GET /diary` without Authorization header
  - **예상 결과:** 401 Unauthorized

- [ ] **1.3.3** 잘못된 JWT 토큰으로 접근 실패
  - **테스트:** `Authorization: Bearer invalid-token`
  - **예상 결과:** 401 Unauthorized

## 3. 보안 테스트 (5개 항목)

### 3.1 인증 및 권한

- [ ] **8.1.1** 인증되지 않은 사용자는 일기 API에 접근 불가
  - **테스트:** 토큰 없이 `GET /diary` 호출
  - **예상 결과:** 401 Unauthorized

- [ ] **8.1.2** 다른 사용자의 일기 접근 불가
  - **테스트:** User A가 User B의 일기 ID로 접근
  - **예상 결과:** 403 Forbidden 또는 404 Not Found

- [ ] **8.1.3** SQL Injection 방지 확인
  - **테스트:** 특수 문자 입력 (`'; DROP TABLE users; --`)
  - **예상 결과:** 정상 처리 또는 유효성 검증 실패, DB 손상 없음

- [ ] **8.1.4** XSS (Cross-Site Scripting) 방지 확인
  - **테스트:** `<script>alert('XSS')</script>` 입력
  - **예상 결과:** 이스케이프 처리되어 표시

- [ ] **8.1.5** CORS 정책 확인
  - **테스트:** 허용되지 않은 Origin에서 API 호출
  - **예상 결과:** CORS 에러
